# Third-party boundary

Ren'Py Live Studio was inspired by the workflow of ActionEditor3:

- https://github.com/kyouryuukunn/renpy-ActionEditor3

The main Live Studio build does not include ActionEditor3's main animation/keyframe implementation. Animation is intentionally disabled and not implemented, unless I can give a better cleaner tool I will not implement it, and I will not be working on doing so until I am fujly happy with the current Live Studio.

Ren'Py itself is licensed separately by its authors. This package is intended to run inside a Ren'Py project and uses Ren'Py's public and runtime APIs.
